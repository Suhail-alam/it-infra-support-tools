# User Onboarding Checklist

Steps I follow when creating and handing over a new user account/device.

- Create user in AD / Azure AD with correct details.
- Assign required groups and licenses.
- Configure mailbox and distribution lists (if needed).
- Prepare laptop/desktop with standard image.
- Join device to domain / Enroll to Intune.
- Apply required security and compliance policies.
- Install required applications for the role.
- Configure Outlook, Teams and VPN.
- Share basic IT usage and security guidelines with the user.
- Update asset management / documentation with device and user details.
