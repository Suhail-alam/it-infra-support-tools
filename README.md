# IT Infrastructure & Support Tools

Collection of small notes and scripts from my work as an IT Infrastructure & Support Engineer.

## What this repo is about

This repo contains things I use or learn in day-to-day IT support:
Windows endpoints, basic networking, Intune/Azure fundamentals and user support tasks.

## Sections

- **Endpoint setup & health checks**  
  Steps and commands for new laptop/desktop setup, health checks and basic troubleshooting.

- **User & access tasks**  
  Notes or small scripts for user onboarding, offboarding and common account issues.

- **Network & VPN troubleshooting**  
  Common checks and commands when users face network or VPN problems.

- **Automation & scripts (learning)**  
  Simple Python / PowerShell scripts where I try to automate repetitive IT support work.

## Tech & tools

- Windows, basic networking
- Intune / Azure fundamentals
- PowerShell / CMD
- Python (for small automations)

## Note

This repo is mainly a reference for my daily IT support work and learning. Over time I will add more scripts, commands and examples.
